This is a simple web application created as part of the DevOps assignment demonstrating version control with Git and GitHub integration.

Project Overview

This project demonstrates:

  • Setting up a local Git repository
  • Using essential Git commands (init, add, commit, branch, etc.)
  • Integration with GitHub for remote version control
  • Basic web application development with HTML

Features

  • Multi-page web application
  • Clean and responsive design
  • Easy navigation between pages
