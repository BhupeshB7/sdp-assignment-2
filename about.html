About This Project

This web application was developed as part of a DevOps course assignment to demonstrate proficiency in version control and containerization technologies.

Assignment Objectives

The primary goals of this project include:

  • Initialize and manage a local Git repository
  • Demonstrate core Git commands and workflows
  • Integrate local repository with GitHub
  • Create and deploy Docker containers
  • Build a simple multi-page web application

Git Commands Demonstrated

  • git init - Initialize repository
  • git status - Check repository status
  • git add - Stage files for commit
  • git commit - Save changes to repository
  • git log - View commit history
  • git branch - Create and manage branches
  • git switch - Switch between branches
